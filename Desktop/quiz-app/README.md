# Quiz Machine – викторина на React

Приложение для проведения викторины с двумя режимами: локальные вопросы на русском языке и внешний API (OpenTDB) с тысячами вопросов на разные темы. Реализовано перемешивание вопросов, подсчёт счёта, переключение темы (светлая/тёмная) и адаптивный дизайн.

## Функциональность
- Два режима: **Локальная** (встроенные вопросы на русском) и **OpenTDB** (внешний API).
- Перемешивание вопросов при каждом запуске.
- Подсчёт правильных ответов и отображение результата (счёт в процентах).
- Подсказки при клике на название режима.
- Адаптивный дизайн для мобильных устройств.
- Эффект «печатной машинки» и неоновая стилистика.

## Стек технологий
- React (функциональные компоненты, хуки)
- CSS3 (кастомные переменные, Flexbox, адаптив)
- OpenTDB API (бесплатный API с вопросами)

## Запуск локально
1. Клонируйте репозиторий:
   git clone https://github.com/Ddaines/quiz-app.git
2. Перейдите в папку проекта:
    cd quiz-app
3. Установите зависимости:
    npm install
4. Запустите приложение:
    npm start
